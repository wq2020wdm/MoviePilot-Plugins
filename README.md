# 🎬 MoviePilot 115 目录洗白整理插件

基于 MoviePilot 内置 115 OpenAPI 及原生 API 开发的极客级网盘整理与增强辅助工具。

从杂乱无章的下载接收文件夹，到完美刮削的海报墙，只需这一个插件。它不仅能为你自动清洗、重命名网盘文件，更内置了强大的**「115 独立 Cookie 逃生舱」**，彻底解决底层 API 阉割导致的离线下载失效问题。

---

## 🌟 核心特性

- **🧹 强力正则洗白 (智能重命名)**：
  - **精准提取番号**：不论是 `[hhd800.com]ABP-123-C.mp4` 还是 `4455@abp_145.mp4`，内置强大的正则表达式引擎都能将其洗白为纯净的 `ABP-145`。
  - **智能标签保留**：自动识别并保留 `-C` / `_C` (中文字幕) 以及 `-UC` / `_UC` (无码破解) 等高价值标签，最终生成如 `ABP-123-C.mp4` 的标准文件名。
  - **父子同名同步**：不仅重命名视频源文件，还会同步重命名最外层的父文件夹，专治各种目录强迫症。

- **🗑️ 自动化垃圾清理**：
  - 根据设定的文件大小阈值（默认 500MB），自动删除影片包里的广告图片、URL 快捷方式、短视频预告等占位文件，只留正片。

- **🚀 终极离线逃生舱**：
  - **无视 API 阉割**：支持填写 115 网页端独立 Cookie，强行接管离线下载通道。
  - **绝对防穿透**：修复了传统离线下载容易“掉入根目录（/云下载）”的 Bug，实现 100% 精准落入你指定的任何多级目录。
  - **全协议支持**：完美解析 `http` / `ftp` / `magnet` / `ed2k` 离线链接。

- **🔗 STRM 存根联动**：
  - 洗白移动完毕后，自动在本地生成 `.strm` 存根，不占本地物理空间，完美对接 Emby / Jellyfin。
  - 生成存根后，可配置自动重启 `mdcx` 等刮削容器，实现“下载 -> 洗白 -> 存根 -> 刮削”的全自动闭环。

- **⚡ 多维触发机制**：
  - 支持 Cron 定时任务、后台面板立即运行，以及 Telegram Bot 指令直接交互。

## 📦 安装与配置

### 1. 插件安装
1. 在 MoviePilot **设置 -> 插件** 中，添加自定义插件仓库：
   `https://github.com/wq2020wdm/MoviePilot-Plugins`
2. 在 **插件市场** 中找到 “115 目录洗白整理” 并在后台完成安装。

### 2. 依赖环境 (⚠️ 重要)
如果你启用了“离线逃生舱”功能，本插件需要依赖 `p115client` 库进行底层通讯。
请确保在你的 MoviePilot 插件目录中，与 `__init__.py` 同级存在 `requirements.txt` 文件，内容为：
```text
p115client
(保存该文件后请 重启 MoviePilot 容器，系统将在启动时静默安装此依赖)

### 3. 设置项指南
在 MoviePilot 的 WEB 设置界面中，推荐按以下指南配置参数：

Cron 表达式: 建议设置为 0 */2 * * * (每两小时执行一次)，避免触发 115 接口风控。

垃圾文件阈值 (MB): 小于该值的文件将被自动剔除，建议设为 500。

监控目录: 你在 115 的下载/接收目录，支持填写多行绝对路径（例如：/接收/小姐姐）。

移动目标路径: 洗白完成后，转移存放的最终归档目录（例如：/影视/电影）。

115云下载目录: 当使用 /cd 离线命令但未指定具体目录时的默认落脚点（例如：/云下载）。

115 独立 Cookie (离线逃生舱): 强烈建议在此处填写从 115 网页端抓取的完整 Cookie，这是突破官方离线下载接口限制的核心。

STRM 内容模板: 默认使用 http://NAS_IP:7811/redirect?path={cloud_file}&pickcode={pick_code}，请按需修改为你的真实直链服务地址。

mdcx 容器名: 填入你的刮削容器名（如 mdcx），留空则生成 STRM 后不触发重启。

🤖 交互指令 (Telegram Bot)
安装并启用后，可在绑定的 Telegram Bot 中发送以下指令：

指令	参数	说明
/run_115_clean	无	立即打断休眠，手动触发一次全量目录扫描与洗白整理任务。
/cd	<链接> [目标目录]	
提交离线下载任务。如不带目录参数，则落入“115云下载目录”。


任务提交成功后，系统将在 5 秒后自动触发一次目录洗白。

(指令示例：/cd magnet:?xt=urn:btih:XXXXXX /接收/最新资源)
