# MoviePilot 115 目录洗白整理插件

基于 MoviePilot 内置 115 OpenAPI 开发的自动化整理工具。

## 🌟 主要功能
- **智能监控**：支持 115 文件夹 ID 或绝对路径（如 `/网盘/下载`）监控。
- **自动洗白**：自动删除文件夹内小于设定阈值（默认 500MB）的文件（清理广告、预告片）。
- **精准重命名**：自动识别文件夹名中的 `@` 符号，保留其后的内容作为新文件名。
- **归档移动**：处理完成后自动将文件夹移动至目标路径，并发送系统通知。
- **多维触发**：支持 Cron 定时任务、MoviePilot 面板立即运行以及 Telegram Bot 指令触发。

## 🚀 安装步骤
1. 在 MoviePilot **设置 -> 插件** 中，添加自定义插件仓库：
   `https://github.com/你的用户名/你的仓库名`
2. 在 **插件市场** 中找到 “115 目录洗白整理” 并安装。
3. 配置监控路径、目标路径及 Cron 表达式。

## 🤖 Bot 指令
安装并启用后，可在 Telegram Bot 中发送以下指令：
- `/run_115_clean`：立即执行一次全量扫描与整理。

## ⚠️ 注意事项
- 请确保 MoviePilot 主程序已正确配置 115 账号。
- 建议 Cron 间隔设置在 30 分钟以上，以避免触发 115 频率限制。
